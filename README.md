# Python: продвинутый уровень
## Кейс: онлайн кинотеатр

- [**Ваня**](https://github.com/Chigatu) — в рамках задания разрабатывает архитектуру решения, описывает и обосновывает методы взаимодействия элементов системы между собой.
-  [**Катя**](https://github.com/Ekaterina3002) — организует ETL-пайплайн для переноса описаний фильмов в соответствующее хранилище.
-  [**Мадина**](https://github.com/madina-zhu) — организует сбор пользовательской активности и потоковое получение файлов из соответствующего хранилища.
- [**Алина**](https://github.com/lesopylka) — организует сбор логов и метрик, а также (в случае выбора микросервисной архитектуры) настраивает распределённый трейсинг.

### Установка зависимостей

```bash
pip install -r requirements.txt --upgrade
```

```bash
cd frontend
npm install
```

### Запуск проекта

Запуск бекенда
``` bash
uvicorn app.main:app --reload
```
Доступен по адресу 'http://127.0.0.1:8000'

Запуска фронта
``` bash
npm run dev
```
Доступен по адресу 'http://localhost:5173'
